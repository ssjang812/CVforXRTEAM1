### 1. 프로젝트명 (가칭)
> Web-cam 비디오 입력을 이용한 Hand Gesture 기반 Virtual Keyboard 구현

### 2. 핵심 목표 (딱 1~3가지)
- A: 웹캠 영상에서 손가락 위치 실시간 인식
Hand Tracking 모델을 이용해 손의 주요 랜드마크 좌표를 추출
검지와 중지 위치를 기반으로 사용자 입력을 인식

- B: 가상 키보드 UI 상호작용
화면에 가상 키보드를 렌더링
손가락 위치가 특정 키 영역에 들어가면 해당 키를 선택

- C: 제스처 기반 키 입력 처리
검지와 중지 사이의 거리 변화를 이용해 클릭 동작을 인식
실제 키보드 입력 또는 화면 상 텍스트 출력으로 결과 확인

### 3. 10일 내 구현 가능성 (Scope Check)
**가장 어려운 기술적 도전:**
- 손가락 랜드마크 좌표와 가상 키보드 버튼 영역을 정확히 매칭하는 로직
- 오입력 방지를 위한 클릭 제스처 조건 설정
**축소 가능한 기능:**
- 키보드 배열을 알파벳 일부만으로 제한
- 실제 키보드 입력 대신 화면 텍스트 출력만 사용
- 한 손 인식만 지원

### 4. 제안 이유
- 컴퓨터 비전에서 가장 직관적인 입력 장치 대체 사례를 다룸
- 손 인식, 좌표 기반 상호작용, UI 시각화를 모두 경험할 수 있음
- 사전학습된 모델을 활용하여 복잡한 학습 과정 없이 구현 가능
- 웹캠 입력과 실시간 결과 출력으로 데모와 발표가 용이함


### 5. 시스템 구성 요약
- 입력: 웹캠 영상

- 처리 과정:
Hand Tracking을 통한 손 랜드마크 검출
검지 좌표와 버튼 영역 비교
손가락 거리 기반 클릭 판별

- 출력:
가상 키보드 시각화
입력된 텍스트 화면 출력
